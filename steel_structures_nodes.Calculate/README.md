# HingedJoint.Calculate — Формулы расчёта РС1

## Обзор

Модуль выполняет анализ узловых соединений стальных конструкций по данным РСУ (расчётные сочетания усилий) и РСН (расчётные сочетания нагрузок), импортированных из SCAD.  
Для каждого критерия строится строка **MAX**, содержащая **все усилия** из одной комбинации нагрузок.

---

## Входные данные

Каждая строка данных содержит усилия из одной комбинации нагрузок:

| Обозначение | Описание |
|---|---|
| **N** | Продольная сила (кН) |
| **Qy** | Поперечная сила по оси Y (кН) |
| **Qz** | Поперечная сила по оси Z (кН) |
| **Mx** | Крутящий момент (кН·м) |
| **My** | Изгибающий момент относительно оси Y (кН·м) |
| **Mz** | Изгибающий момент относительно оси Z (кН·м) |
| **Mw** | Бимомент (кН·м²) |

Несущая способность узла (из альбома):

| Обозначение | Описание |
|---|---|
| **Q_cap** | Несущая способность по поперечной силе Qy (кН) |
| **N_cap** | Несущая способность по растяжению Nt (кН) |
| **psi** | Коэффициент psi из альбома несущей способности |

---

## Алгоритм построения таблицы анализа

### Шаг 1 — Объединение данных

Все строки РСУ и РСН объединяются в единый пул:

$$
\text{Pool} = \text{RSU} \cup \text{RSN}
$$

где:
- $\text{Pool}$ — объединённый массив всех строк усилий
- $\text{RSU}$ — строки расчётных сочетаний усилий (из SCAD)
- $\text{RSN}$ — строки расчётных сочетаний нагрузок (из SCAD)

### Шаг 2 — Фильтр по элементам

Пользователь вводит в текстовое поле номера конечных элементов (КЭ) из модели SCAD, примыкающих к рассматриваемому узлу.  
Допустимые форматы ввода: `73-80`, `73...80`, `73 74 75`, `73, 74; 75` или их комбинации.  
Диапазоны автоматически разворачиваются в отдельные номера: `73-80` → `73, 74, 75, 76, 77, 78, 79, 80`.

Если поле заполнено — из пула оставляются только строки, где номер элемента входит в указанный набор:

$$
\text{Elem}_i \in E_{filter}
$$

где:
- $\text{Elem}_i$ — номер конечного элемента в i-й строке данных
- $E_{filter}$ — набор номеров КЭ, введённый пользователем в поле «Фильтр элементов»

Если поле пустое — используются все элементы без фильтрации.

### Шаг 3

Оставляются только строки с минимальным номером сечения (сечение у узла):

$$
\text{Sect} = \min(\text{Sect}_i)
$$

где:
- $\text{Sect}$ — номер сечения, оставляемого в выборке (минимальный = ближайший к узлу)
- $\text{Sect}_i$ — номер сечения в i-й строке

### Шаг 4

Для каждого критерия выбирается строка **целиком** (все усилия из одной комбинации).

---

## Формулы выбора MAX-строк

### 1. MAX N — Максимальная продольная сила

Выбирается строка с наибольшим абсолютным значением продольной силы:

$$
\text{MAX N}: \quad \max \lvert N_i \rvert
$$

где:
- $N_i$ — продольная сила в i-й строке (кН)
- $i$ — индекс строки (комбинации нагрузок)

### 2. MAX N+

Выбирается строка с наибольшим положительным значением $N$:

$$
\text{MAX N+}: \quad N_t = \max (N_i), \quad N_i > 0
$$

где:
- $N_t$ — максимальная растягивающая сила (кН)
- $N_i$ — продольная сила в i-й строке; учитываются только положительные значения (растяжение)

### 3. MAX N-

Выбирается строка с наименьшим (наибольшим по модулю) отрицательным значением $N$:

$$
\text{MAX N-}: \quad N_c = \min (N_i), \quad N_i < 0
$$

где:
- $N_c$ — максимальная сжимающая сила (кН, отрицательное значение)
- $N_i$ — продольная сила в i-й строке; учитываются только отрицательные значения (сжатие)

### 4. MAX Qy — Максимальная поперечная сила Qy

$$
\text{MAX Qy}: \quad \max \lvert Qy_i \rvert
$$

где:
- $Qy_i$ — поперечная сила по оси Y в i-й строке (кН)
- $i$ — индекс строки (комбинации нагрузок)

### 5. MAX Qz

$$
\text{MAX Qz}: \quad \max \lvert Qz_i \rvert
$$

где:
- $Qz_i$ — поперечная сила по оси Z в i-й строке (кН)
- $i$ — индекс строки (комбинации нагрузок)

### 6. MAX Mx

$$
\text{MAX Mx}: \quad \max \lvert Mx_i \rvert
$$

где:
- $Mx_i$ — крутящий момент в i-й строке (кН·м)
- $i$ — индекс строки (комбинации нагрузок)

### 7. MAX My

$$
\text{MAX My}: \quad \max \lvert My_i \rvert
$$

где:
- $My_i$ — изгибающий момент относительно оси Y в i-й строке (кН·м)
- $i$ — индекс строки (комбинации нагрузок)

### 8. MAX Mz

$$
\text{MAX Mz}: \quad \max \lvert Mz_i \rvert
$$

где:
- $Mz_i$ — изгибающий момент относительно оси Z в i-й строке (кН·м)
- $i$ — индекс строки (комбинации нагрузок)

### 9. MAX Mw

$$
\text{MAX Mw}: \quad \max \lvert Mw_i \rvert
$$

где:
- $Mw_i$ — бимомент в i-й строке (кН·м²)
- $i$ — индекс строки (комбинации нагрузок)

---

## Формула коэффициента использования U

### 10. MAX Coeff / MAX u — Максимальный коэффициент использования

Для каждой строки вычисляется коэффициент использования:

$$
U_i = \frac{\lvert Qz_i \rvert}{Q_{cap}} + \frac{\lvert N_i \rvert}{N_{cap}}
$$

где:
- $U_i$ — коэффициент использования несущей способности для i-й комбинации нагрузок (безразмерный)
- $Qz_i$ — поперечная сила по оси Z в i-й строке (кН)
- $Q_{cap}$ — предельная несущая способность узла по поперечной силе (AlbumQy) из альбома (кН)
- $N_i$ — продольная сила в i-й строке (кН)
- $N_{cap}$ — предельная несущая способность узла по растяжению (AlbumNt) из альбома (кН)

Выбирается строка с максимальным значением $U$:

$$
\text{MAX u}
$$

где:
- $U_{max}$ — наибольший коэффициент использования среди всех комбинаций

> Формула применяется только при $Q_{cap} \neq 0$ и $N_{cap} \neq 0$.

---

## Разделение N на Nt и Nc

Для каждой выбранной MAX-строки продольная сила разделяется:

$$
N_t = \begin{cases} N, & \text{если } N > 0 \\ 0, & \text{иначе} \end{cases}
$$

$$
N_c = \begin{cases} N, & \text{если } N < 0 \\ 0, & \text{иначе} \end{cases}
$$

где:
- $N$ — продольная сила из выбранной MAX-строки (кН)
- $N_t$ — растягивающая составляющая: равна $N$ если $N > 0$, иначе 0
- $N_c$ — сжимающая составляющая: равна $N$ если $N < 0$, иначе 0

---

## Вспомогательные функции

### MaxAbs — Максимум по модулю

$$
\text{MaxAbs}(v_1, \ldots, v_n) = \max \lvert v_i \rvert
$$

где:
- $v_1, \ldots, v_n$ — набор значений усилия (одного типа) по всем комбинациям
- Результат — наибольшее значение по модулю

### MaxPositive

$$
\text{MaxPositive}(v_1, \ldots, v_n) = \max(v_i \mid v_i > 0)
$$

где:
- $v_i$ — значения усилия; учитываются только положительные
- Результат — наибольшее положительное значение (растяжение)

### MinNegative

$$
\text{MinNegative}(v_1, \ldots, v_n) = \min(v_i \mid v_i < 0)
$$

где:
- $v_i$ — значения усилия; учитываются только отрицательные
- Результат — наименьшее отрицательное значение (максимальное сжатие)

## Математическая запись формул

### Обозначения

Пусть дано $n$ строк данных (комбинаций нагрузок) после фильтрации.  
Каждая $i$-я строка содержит вектор усилий:

$$
\mathbf{S}_i = (N_i,\; Qy_i,\; Qz_i,\; Mx_i,\; My_i,\; Mz_i,\; Mw_i), \quad i = 1, \ldots, n
$$

где:
- $\mathbf{S}_i$ — вектор всех усилий из i-й комбинации нагрузок
- $n$ — общее количество комбинаций нагрузок после фильтрации
- $N_i$ — продольная сила (кН)
- $Qy_i, Qz_i$ — поперечные силы по осям Y и Z (кН)
- $Mx_i$ — крутящий момент (кН·м)
- $My_i, Mz_i$ — изгибающие моменты (кН·м)
- $Mw_i$ — бимомент (кН·м²)

Несущая способность узла из альбома:

$$
Q_{cap} = \text{AlbumQy}, \quad N_{cap} = \text{AlbumNt}, \quad \psi = \text{AlbumPsi}
$$

где:
- $Q_{cap}$ — предельная поперечная сила узла из альбома (кН)
- $N_{cap}$ — предельная растягивающая сила узла из альбома (кН)
- $\psi$ — коэффициент $\psi$ из альбома несущей способности
- $\text{AlbumQy, AlbumNt, AlbumPsi}$ — названия полей в JSON-файле альбома

---

### Фильтрация данных

**Объединение источников:**

$$
\mathcal{D} = \mathcal{D}_{RSU} \cup \mathcal{D}_{RSN}
$$

где:
- $\mathcal{D}$ — полный набор данных
- $\mathcal{D}_{RSU}$ — данные расчётных сочетаний усилий
- $\mathcal{D}_{RSN}$ — данные расчётных сочетаний нагрузок

**Фильтр по элементам**

Пользователь задаёт номера КЭ в текстовом поле (например: `73-80`, `73 74 75`).  
Диапазоны разворачиваются: `73-80` → `73, 74, 75, 76, 77, 78, 79, 80`.  
Если поле заполнено — оставляются только строки с совпадающим номером элемента:

$$
\mathcal{D}' = \lbrace \mathbf{S}_i \in \mathcal{D} \mid \text{Elem}_i \in E_{filter} \rbrace
$$

где:
- $\mathcal{D}'$ — отфильтрованный набор данных
- $\text{Elem}_i$ — номер конечного элемента (КЭ) в i-й строке
- $E_{filter}$ — набор номеров КЭ из поля «Фильтр элементов» (если пустое — фильтрация не применяется)

**Фильтр по сечению**

$$
s^* = \min \text{Sect}_i, \qquad \mathcal{D}'' = \lbrace \mathbf{S}_i \in \mathcal{D}' \mid \text{Sect}_i = s^* \rbrace
$$

где:
- $s^*$ — минимальный номер сечения (ближайшее к узлу)
- $\text{Sect}_i$ — номер сечения в i-й строке
- $\mathcal{D}''$ — итоговый набор данных после всех фильтров

---

### Выбор MAX-строк

Для каждого усилия $F \in \lbrace N, Qy, Qz, Mx, My, Mz, Mw \rbrace$:

$$
k_F = \max \lvert F_i \rvert, \qquad \text{MAX } F = \mathbf{S}_{k_F}
$$

где:
- $F$ — тип усилия ($N$, $Qy$, $Qz$, $Mx$, $My$, $Mz$ или $Mw$)
- $F_i$ — значение усилия $F$ в i-й строке
- $k_F$ — индекс строки с наибольшим модулем усилия $F$
- $\mathbf{S}_{k_F}$ — полный вектор усилий из этой строки (все 7 компонентов)

Отдельно для продольной силы

$$
k_{N^+} = \max (N_i),\; N_i > 0, \qquad \text{MAX } N^{+} = \mathbf{S}_{k_{N^+}}
$$

$$
k_{N^-} = \min (N_i),\; N_i < 0, \qquad \text{MAX } N^{-} = \mathbf{S}_{k_{N^-}}
$$

где:
- $k_{N^+}$ — индекс строки с наибольшим положительным $N$ (максимальное растяжение)
- $k_{N^-}$ — индекс строки с наименьшим отрицательным $N$ (максимальное сжатие)
- $\mathbf{S}_{k}$ — полный вектор усилий из найденной строки

---

### Разделение N на Nt и Nc

$$
N_t^{(i)} = \max(N_i,\; 0) = \begin{cases} N_i, & N_i > 0 \\ 0, & N_i \leq 0 \end{cases}
$$

$$
N_c^{(i)} = \min(N_i,\; 0)
$$

где:
- $N_t^{(i)}$ — растягивающая составляющая продольной силы для i-й строки (кН)
- $N_c^{(i)}$ — сжимающая составляющая продольной силы для i-й строки (кН)
- $N_i$ — исходная продольная сила в i-й строке (кН)

---

### Коэффициент использования

Для каждой $i$-й строки:

$$
U_i = \frac{\lvert Qz_i \rvert}{Q_{cap}} + \frac{\lvert N_i \rvert}{N_{cap}}, \qquad Q_{cap} \neq 0,\; N_{cap} \neq 0
$$

где:
- $U_i$ — коэффициент использования для i-й комбинации (безразмерный)
- $Qz_i$ — поперечная сила по оси Z в i-й строке (кН)
- $N_i$ — продольная сила в i-й строке (кН)
- $Q_{cap}$ — предельная поперечная сила узла из альбома (кН)
- $N_{cap}$ — предельная растягивающая сила узла из альбома (кН)

Наиболее нагруженная комбинация:

$$
k_U = \max U_i, \qquad U_{max} = U_{k_U}, \qquad \text{MAX } u = \mathbf{S}_{k_U}
$$

где:
- $k_U$ — индекс строки с наибольшим коэффициентом использования
- $U_{max}$ — наибольший коэффициент использования
- $\mathbf{S}_{k_U}$ — полный вектор усилий из самой нагруженной комбинации

---

### Вспомогательные функции

$$
\text{MaxAbs}(v_1, \ldots, v_n) = \max \lvert v_i \rvert
$$

где:
- $v_1, \ldots, v_n$ — значения одного усилия по всем комбинациям нагрузок
- Результат — наибольшее значение по модулю (без учёта знака)

$$
\text{MaxPositive}(v_1, \ldots, v_n) = \max \lbrace v_i \mid v_i > 0,\; i = 1, \ldots, n \rbrace
$$

где:
- $v_i > 0$ — из набора берутся только положительные значения
- Результат — наибольшее положительное значение (максимальное растяжение)

$$
\text{MinNegative}(v_1, \ldots, v_n) = \min \lbrace v_i \mid v_i < 0,\; i = 1, \ldots, n \rbrace
$$

где:
- $v_i < 0$ — из набора берутся только отрицательные значения
- Результат — наименьшее отрицательное значение (максимальное сжатие)

---

### Сводка экстремальных значений

Из построенных MAX-строк формируется итоговый вектор:

$$
N_{sum} = N_{k_N}, \quad N_{t,sum} = N_{k_{N^+}}, \quad N_{c,sum} = N_{k_{N^-}}
$$

где:
- $N_{sum}$ — продольная сила из строки MAX $N$ (кН)
- $N_{t,sum}$ — максимальная растягивающая сила из строки MAX $N^+$ (кН)
- $N_{c,sum}$ — максимальная сжимающая сила из строки MAX $N^-$ (кН)
- $k_N, k_{N^+}, k_{N^-}$ — индексы соответствующих MAX-строк

$$
Qy_{sum} = \lvert Qy_{k_{Qy}} \rvert, \quad Qz_{sum} = \lvert Qz_{k_{Qz}} \rvert
$$

где:
- $Qy_{sum}$ — модуль поперечной силы $Qy$ из строки MAX $Qy$ (кН)
- $Qz_{sum}$ — модуль поперечной силы $Qz$ из строки MAX $Qz$ (кН)

$$
Mx_{sum} = \lvert Mx_{k_{Mx}} \rvert, \quad My_{sum} = \lvert My_{k_{My}} \rvert, \quad Mz_{sum} = \lvert Mz_{k_{Mz}} \rvert, \quad Mw_{sum} = \lvert Mw_{k_{Mw}} \rvert
$$

где:
- $Mx_{sum}$ — модуль крутящего момента из строки MAX $Mx$ (кН·м)
- $My_{sum}, Mz_{sum}$ — модули изгибающих моментов из строк MAX $My$ и MAX $Mz$ (кН·м)
- $Mw_{sum}$ — модуль бимомента из строки MAX $Mw$ (кН·м²)

$$
U_{sum} = U_{max}, \qquad \psi_{sum} = \psi
$$

где:
- $U_{sum}$ — наибольший коэффициент использования несущей способности (безразмерный)
- $\psi_{sum}$ — коэффициент psi из альбома несущей способности (безразмерный)
